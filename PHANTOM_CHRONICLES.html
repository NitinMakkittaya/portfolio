Case File #002: THE PHANTOM CHRONICLES

III. INVESTIGATOR'S SUMMARY

The objective was to deconstruct reality in real-time. We needed a digital laboratory capable of isolating a human subject from their environment without physical hardware like green screens. I engineered a live-stream VFX interface that uses deep learning to segment the subject from the background, allowing for instantaneous environment swapping and the application of reality-bending visual filters.

IV. EVIDENCE LOG (Tech Stack)

  • The Brain: Python, rembg (U2-Net Deep Learning model)
  • Visual Processing: OpenCV (cv2), NumPy (Matrix manipulation)
  • The Interface: Streamlit
  • Imaging Pipeline: Pillow (PIL), Imageio

V. MODUS OPERANDI (How It Works)

The investigation utilizes a multi-layered processing pipeline to alter the visual record:

  • Surveillance Feed: The system hijacks the webcam hardware via OpenCV, capturing a continuous BGR image stream.
  • Saliency Segmentation: Each frame is fed into the AI engine to generate a precise 8-bit alpha mask, isolating the subject.
  • Environment Synthesis: Pixels behind the mask are dynamically replaced with Gaussian blurs or synthetic color fields.

Once the subject is isolated, the visual data is corrupted using mathematical filters: Lateral R-G-B channel shifting for "Glitch" effects and Canny Edge Detection for "Neon" tracing.

VI. THE VERDICT

The application delivers a high-fidelity, interactive video stream that proves identity can be isolated from its surroundings. It enables users to exist in a VIRTUAL STUDIO with professional-grade background removal processed entirely on-the-fly.

VII. CASE ASSETS

  • Pre-trained Protocol: Utilizes the U2-Net model, eliminating the requirement for physical chroma-key hardware.
  • Matrix Vectorization: Employs NumPy to apply filters across millions of pixels simultaneously without stalling the live feed.
  • Dynamic Command Center: A sidebar-controlled dashboard for instantaneous toggling of reality parameters.
